# WhatsApp Google Drive Assistant (Task 2)

This is a simple n8n workflow for my internship task with **Think Act Rise Foundation**.

### ✅ Features Implemented
- Receives WhatsApp messages via Twilio Webhook
- Detects if message contains `"upload"`
- Sends back a custom reply using Set node
- Workflow designed and built in **n8n cloud**, without Docker

### 💡 Next Steps
- Will connect to Google Drive for file search/upload
- Will handle other message types (like "list" or "help")
