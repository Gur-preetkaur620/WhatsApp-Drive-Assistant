# WhatsApp Google Drive Assistant (Task 2)

This repository is part of the internship assignment from *Think Act Rise Foundation*.

## 📌 Task Description (Task 2)
> Build a WhatsApp assistant using n8n that connects with Google Drive.  
> The assistant should be able to receive messages and perform basic operations like replying, listing files, or uploading.

---

## ✅ What Has Been Done
- 📲 WhatsApp Webhook connected via Twilio Sandbox
- ✅ Incoming messages are received in n8n
- 🔎 A condition is checked using IF node ("upload")
- 💬 Auto-reply is sent using Set node if message contains upload
- ✅ Workflow built and tested successfully (no Docker used)
